# Windows Active Directory Home Lab

## Overview
This project is a full Active Directory home lab built using VirtualBox. It simulates a real enterprise IT environment including domain services, user management, Group Policy configuration, file sharing, and helpdesk troubleshooting.

---

## Lab Progression

- LAB 0: Virtual Environment Setup (VirtualBox, Windows Server, Windows 11)
- LAB 1: Windows Server Installation
- LAB 2: Domain Controller Setup
- LAB 3: Active Directory (Users, Groups, OUs)
- LAB 4: Join Windows 11 Client to Domain
- LAB 5: User Management (Password Resets, Account Lockouts)
- LAB 6: Group Policy Objects (GPO)
- LAB 7: File Sharing & Permissions
- LAB 8: Helpdesk Ticket Simulation
- LAB 9: Documentation & Resume Preparation

---

## Skills Demonstrated
- Windows Server administration
- Active Directory configuration
- Domain controller setup
- User and group management
- Group Policy configuration
- File sharing and NTFS permissions
- IT troubleshooting (Tier 1 helpdesk scenarios)
- Technical documentation

## Purpose
This project demonstrates practical IT support and system administration skills through a fully simulated enterprise environment.
